# Etch-A-Sketch

Simple web application for drawing.
<br> <br>

  
  
### Grid  
Set the number of squares per side for the new grid.  
<br>

### Random  
Every square takes a random color.  
<br>

### Shadow  
Progressively makes the color more black.  
<br>

### Border  
Removes or adds a border for each square.  
<br>

### Clear  
Clears the drawing.
